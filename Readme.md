
# github-growl

Cross-platform desktop notifications for your Github repositories.

Currently supports Mac OS X (Darwin), Ubuntu (Linux), and Windows.

## Quick start

```bash
npm install -g github-growl
```

## Example Usage

```bash
cd ~/my-git-project
github-growl --user niftylettuce --token 1234567890
```
